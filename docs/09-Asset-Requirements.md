# Asset Requirements

This document tracks all 3D geometries, texture files, video/image media, resumes, certificates, and fonts needed for the portfolio.

---

## 1. 3D Model Assets
- **Avatar Base Mesh**
  - Format: `.glb` / `.gltf`
  - Specifications: Low-to-mid poly (optimized for web, <15MB, DRACO compressed).
- **Clothing & Accessories**
  - Hoodie model (with detailed PBR map, normal map for fabric texture).
  - ID Badge and Lanyard.
  - Tablet model.
  - Document/Archive folder.
- **Environment Props**
  - Subtle studio backdrop (minimal walls, floor texture, ambient shadow catchers).

---

## 2. Textures & Materials
- **Fabric Texture**: Albedo, Normal, and Roughness maps (1K or 2K resolution) for the hoodie.
- **Metal Texture**: For the zipper and tablet casing.
- **Paper Texture**: For certificates and documents.
- **Screen Texture**: UI mockups for the tablet screen interface.

---

## 3. 2D & Video Media (`assets/`)
- **Images (`assets/images/`)**
  - Project screenshots.
  - Headshots/Avatar face textures.
  - Fallback background images.
- **Videos (`assets/videos/`)**
  - Showcase reels for Neura Sentinel, Founder Finder, etc.
- **Certificates (`assets/certificates/`)**
  - High-res images or PDFs of completed certifications.
- **Resumes (`assets/resumes/`)**
  - PDF formatted resumes for recruiter downloads.

---

## 4. Typography & Fonts
- **Display Font**: Outfit, Inter, or Space Grotesk (served via Google Fonts).
- **Body Font**: Inter or Roboto.
